 # Java Clean Code
 
- 클래스 이름( 명사, 명사구 )

- 메서드 이름( 동사, 동사구 )

- 블록과 들여쓰기
  - 1단 혹은 2단을 넘지마라!

- 명령과 조회를 분리하라
  - 뭔가를 수행하거나 반환하거나 둘 중 하나만 해라!
  - 오류처리도 한 가지 작업이다!

- 주석
  - 법적인 주석
  - 공식이나 정보를 제공하는 주석

- null을 전달, 반환하지 마라
 
- Log4j 익히기( Logger.log(e.getMessage))
- 변수명 정하기
  - 해법영역
  - 문제영역
